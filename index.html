Lead Qualification
ICP checklist, qualification questions, and inbound rules.
Inbound Lead Checklist
  • 🏢
    Tier 1 or Tier 2 account? — If not, don't book
  • 💻
    Company develops software (not services)? — If they offer software development services, don't book
  • 🌍
    HQ in right territory? — No LATAM, APAC, EMEA, Middle East, India, Iran
  • ⚡
    No existing S3/S4/S5 opportunity? — Check Triforce first
  • 👤
    ICP persona? — Director and above. No SAP/ABAP/S4HANA, no Low Level (managers, engineers), no consultants
  • 🏭
    Valid industry? — Not: Business Services, Education, Churches, Non-Profit, Consultants, Government
  • 📧
    Not a Tricentis employee? — No @tricentis.com email
  • 📊
    SeaLights ARR is 0 or blank? — If they're already a customer, check with Adi
Qualification Questions to Ask on Call
  • 🔤
    Programming languages — "What are the primary programming languages used in your inhouse applications?"
  • 🏗️
    Inhouse vs outsourced — "They are developed inhouse right? Nothing outsourced?"
  • ⏱️
    Primary interest — "Would you be looking to reduce time? Improve quality? Reduce the cost of testing altogether?"
  • ⏳
    Test cycle time — "How long does it usually take to run a full suite of E2E or regression tests?"
  • 🔁
    Run all tests? — "Do you run ALL of your tests ALL the time?"
  • 🛡️
    Escaping defects — "Do you have issues with escaping defects or untested code reaching production?"
  • 🏢
    Team structure — "Do you have a separate QA team or do engineers handle QA?"
